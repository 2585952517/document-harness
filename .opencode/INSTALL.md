# OpenCode Installation

Install this repository as an OpenCode plugin package. The package entrypoint is `.opencode/plugins/document-harness.mjs`.

The plugin adds this repository's `skills/` directory to OpenCode config and injects a short session reminder to use `harness-author` for conversational `harness.yaml` creation, `promission-supervisor` for `document-harness.lock` promission supervision, `memory/project-understanding.md` for project essence memory, and `agents/maintainer.md` for delegated Document Harness maintenance.
